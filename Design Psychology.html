Human–Device Interaction · Trauma-Informed Design

Design Psychology — a regulated surface for a dysregulated state

The Atlas addresses a user in a plausibly dysregulated state — autonomically aroused by exam stress, fatigued, traumatised, or neurodivergent. Interaction is treated as nervous-system design: every device behaviour is evaluated for the regulatory load it imposes before any learning is attempted.

The Thesis

Minimise the regulatory load — autonomic, attentional, cognitive — that a stressed or atypical nervous system must absorb before it can engage the material.

LAYER 01Ambient — the pre-attentive environment

Before any deliberate interaction, the rendered surface is appraised by the threat-detection system. These properties are tuned to be read as safe pre-attentively — beneath conscious notice.

A1

Spatial openness, not enclosure

OPEN · room to breathe
StressorEnclosure and visual density are common interoceptive triggers for anxiety and PTSD; a dense content field is appraised as a narrowing space and raises autonomic arousal.
ResponseCounter it spatially — an open, horizon-like field with generous negative space, so the container is appraised as expansive rather than confining.
Safety. The autonomic system down-regulates before the cognitive system is asked to engage.
A2

Familiarity priming before demand

StressorThe first rendered object calibrates threat appraisal; a novel, complex stimulus primes intimidation and defensive arousal.
ResponsePopulate the field with erythrocytes — the most over-learned object in medicine. Fluent recognition signals competence pre-consciously, lowering the threat estimate before any content is parsed.
Priming. Establish mastery, not difficulty, as the first impression.
A3

User-initiated motion, never system-initiated

Acts on you
Moves on its own. A startle you can't predict.
Responds to you
StressorSystem-initiated motion is an event that acts on the user — unpredictable, and for a hypervigilant or dissociating nervous system, a potential startle or trigger.
ResponseAll motion is input-coupled: the scene advances only as a function of user scroll. The device responds; it never initiates. The same drift then functions as a grounding signal — temporal orientation held by the user.
Agency. Relocating the locus of control from device to user is itself regulating. Drag the right panel — nothing occurs until you cause it.
A4

Circadian-affective grading

ResponseThe full surface is graded along a teal → golden-hour axis — a circadian-affective register reliably associated with low arousal and perceived safety.
Both trajectories — day→dusk and midnight→dawn — resolve toward warmth; the affective arc always terminates in a safer state than it began.

LAYER 02Agency — user-governed rendering

Trauma-informed practice centres voice and choice; HDI frames the same construct as locus of control. The Atlas exposes its own rendering as a settings surface, so the user — not the designer — determines what the device does. Perceived control is itself anxiolytic.

AG1

From barebones to alive — the user sets the floor

StressorNo single level of ambient animation suits every nervous system; what soothes one user over-stimulates another, and the designer cannot predict which.
ResponseExpose every ambient layer as a plain-language control — Show cells, React to scrolling, Hold still, Colour gradient. The user dials from a static, barebones surface to a continuously alive one, with no jargon to decode.
Voice & choice. The regulating party is the user, not the interface.
AG2

A stillness floor — and what is deliberately not exposed

Hold stillfreezes all motion
Colour gradient offflat black / white
Focus modeone system, minimal chrome
Diffusion / legibilitynot user-exposed
FloorCombined, Hold still, gradient off, and Focus mode reduce the surface to its lowest-stimulation state — a near-static, single-system, flat-background reading view.
BoundaryOne parameter is withheld: the diffusion layer that keeps text legible over the scene. Exposing it would let a user degrade their own readability — a control whose only outcomes are harmful is not offered.
Bounded autonomy. Agency over affect and stimulation; never over legibility or safety.
LAYER 03Interaction — the event model

One invariant governs the event model: the user initiates; the device responds. Disclosure is opt-in, attention is single-focus, actions are reversible, and stressors are never imposed.

B1

Progressive disclosure — nothing rendered until requested

StressorRendering more than fifty resources across thirteen systems simultaneously presents a high-density threat surface with no visual point of rest.
ResponseAll branches collapse by default. The interface presents a short set of system labels; the user titrates their own exposure, deciding how much information enters the field and when.
Self-pacing. The optimisation target is non-overwhelm, not engagement — a deliberate inversion of the conventional metric.
B2

Single-focus attention model

Click a row — watch the others recede.
StressorUnder autonomic load, parallel decision sets exceed available attentional bandwidth.
ResponseSelecting a system foregrounds it and recedes the rest. The architecture enforces one active locus; competing options are suppressed, not removed. In Focus mode the surface renders that single system alone.
Attentional economy. De-cluttering by suppression, not deletion.
B3 · B4

Reversibility and opt-in stressors

Exam countdown — off by default
ReversibilityEvery resource opens with a persistent return control and honours native back-navigation. State is always reversible; the user is never trapped in a view — the interaction equivalent of a visible exit.
Opt-inThe exam countdown is disabled by default. Rather than imposing a salient stress cue, the system requires the user to instantiate it. Temporal pressure is opt-in.
Trustworthiness. The device introduces no stressor the user did not choose.
LAYER 04Cognitive — knowledge representation

The same load-minimisation applied to the information itself: lower the working-memory cost of comprehension before it is demanded.

C1

Externalising sequence — offloading working memory

StressorProse requires the reader to maintain antecedent clauses in working memory while parsing subsequent ones — taxing the precise resource depleted by stress, fatigue, ADHD, or autism.
ResponseRe-represent narrative as a directed graph. The arrows externalise sequence onto the display; order is held by the page, not the reader. Text is retained only where it outperforms structure.
Dual coding. Recruiting the visuospatial channel alongside the verbal distributes load across two subsystems rather than one bottleneck.
C2

Bounded units — one chunk at a time

open
ResponseKnowledge is chunked into self-contained units — system → disease → a single graph. The user enters a bounded scope, completes it, and exits.
One disease's logic is reasoned about without co-activating the entire domain; never more than one bounded unit is resident at a time.
Chunking. The working-memory thesis at the scale of the knowledge map.

05One thesis, four layers

The layers are one construct expressed across four media. None contradicts the others — that coherence is the claim.

Ambient
  • Spatial openness vs. enclosure
  • Familiarity primes competence
  • User-initiated motion only
  • Circadian-affective grading
Agency
  • Plain-language rendering controls
  • Barebones → alive, user sets it
  • Stillness & flat-background floor
  • Bounded autonomy (legibility withheld)
Interaction
  • Progressive disclosure by default
  • Single-focus attention model
  • Reversible state, visible exit
  • Opt-in stressors only
Cognitive
  • Sequence externalised to graphs
  • Dual coding — words + space
  • Bounded chunks, one at a time
  • Non-essential signal suppressed

Four layers — ambient, agency, interaction, cognitive — each removing one class of load between a dysregulated nervous system and the material it must engage. That is the design.

Regulation is not an aesthetic here. It is the specification.

A note on rigour: these choices are trauma-informed and grounded in established constructs — locus of control and perceived control (HDI), progressive disclosure, cognitive-load theory, dual coding, and the SAMHSA trauma-informed principles of safety, trustworthiness, and choice & control. They are offered as a coherent design rationale, not a clinically validated intervention; intent, coherence, and care are the claims. Everything here reflects the author's personal opinions and interpretation — a point of view, not professional, medical, or psychological advice, and no particular outcome is promised.
